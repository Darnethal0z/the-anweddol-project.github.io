The Anweddol server implementation is using a YAML file for configuration.

It is stored in :

  • ~/.anweddol/rsa_keys on linux ;

  • C:\\Users\%USERNAME%\Anweddol\rsa_keys on windows ;

after installation, read its header to learn about it.
