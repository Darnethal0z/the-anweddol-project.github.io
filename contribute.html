Contribute to the project

We are always in search of new ideas and enhancements to improve the Anweddol project. Here is a non-exhaustive list of ways to help for the cause :

  • Maintain / Test / Review the source code

    The Anweddol implementations source code are mostly written in Python. If you want to contribute, see the contribution guidelines of the affiliated repositories.

  • Report bugs / errors

    Nothing is perfect, but everything can be improved. If you see an error or an unexpected behaviour during the usage of one of the implementations, you can report it on the affiliated repository issues page, and feel free to propose corrections or alternatives.

  • Bring new ideas and enhancements

    Anweddol being a young project, it still needs some improvements and new features to be enhanced: You can bring you own ideas regardless of the domain on the several platforms that you can retrieve on the [Social] section.

  • Promote the cause

    The project's cause must be widely spreaded in order to be a healthy and reliable one : Any kind of promotions are welcome. If you want to promote the cause, you are free to use any assets provided [here] and on this website.

  • Assist other users

    One of the characteristics of a community-driven project is that its members can help each other to alleviate the difficulties they may encounter when using one of the Anweddol implementations. Peoples who are familiar with Anweddol can be of assistance on help forums or miscellaneous platforms.

  • Your idea ?

    Have another idea ? Let the community know on the several Anweddol community platforms; see the [Social] section to learn more.
