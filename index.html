Open-source and community-driven software

Written in Python, the software has been designed to be versatile and flexible.
You can make a completely new client or server implementation from scratch, integrate it into on an existing project, customize your own application ... the choice is yours.

Getting started

Anweddol is actually separated in 3 programs :

  • The client implementation : It is the software that clients use to interact with Anweddol servers.
  • The server implementation : It is the actual server software providing the temporary virtual machines to the clients.

For a proper introduction on how Anweddol is made, see the [Documentation section] to learn more.
