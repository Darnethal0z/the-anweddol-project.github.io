-- The Anweddol Project --

Anweddol is a server implementation providing temporary, SSH-controllable virtual machines across the web. The software is still under development : this website provides quick informations about it. Please note that some of the technical specifications presented on this site now can be different, or even absent in the final version.

- Presentation -

It is a server system running temporary virtual machines that can be remotely controlled by a client, and destroyed once used.
