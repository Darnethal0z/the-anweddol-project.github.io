Installation

This section is dedicated to the Anweddol implementation installation. See the articles below to know how to install them on your system.

Anweddol-client implementation

Clone the repository :

$ git clone https://github.com/the-anweddol-project/Anweddol-client.git
$ cd Anweddol-client

And install it with pip :

Linux$ pip install . Windows$ py -m pip install .

Read the [Usage guide] to get a proper introduction, you can also see the [Client developer section] about Anweddol client API bindings.

Anweddol-server implementation

The Anweddol server installation requires a specific setup before installation and usage in order to provide a functional service.

See the server [Administration guide] to learn more.
